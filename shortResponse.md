## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?

Describe what makes a unit responsive and why that matters when building a website.

static unit like `px` always going to be that value that we wrote no matter on what screen we sit.
responsive unit adapt to your view port and always stays on every screen with what value you want.

## Question 2 — Explain

Look at these two CSS rules:

```css
.image {
  width: 400px;
}

.image {
  width: 50%;
}
```

Explain what happens to the image on a small screen with each rule. Why does one behave better than the other?

image with `px` value will always be 400px on every screen and if we look at the picture in chromebook we will see normal picture but on the phone this picture will be so big because phone screen is smaller that chromebook's screen.
If we use `%` value which is responsible this picture will can adapt to your screen and can be normal on every screen
